## Key honest limitations

1. **FIFA ends 2024.** The system must not invent 2026 FIFA values; the as-of
   join returns the last (stale) release with `days_since_release` so callers
   can refuse or down-weight it.
2. **Transfermarkt is club football.** It enriches player *club* form/value,
   not international fixtures (670 national-team games only).
3. **No StatsBomb data** locally — event/xG features are blocked until provided.
4. **API-Football Free plan** cannot serve 2026 WC fixtures/lineups; a paid
   plan would be required. Provider predictions are benchmark-only.
